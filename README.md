# ♔ Chess Pro - لعبة الشطرنج الاحترافية

لعبة شطرنج احترافية كاملة بتصميم كلاسيكي فخم مع ذكاء اصطناعي قوي جداً.

## 🎮 الميزات الرئيسية

### محرك الشطرنج الكامل
- ✅ جميع قواعد الشطرنج الحقيقية
- ✅ الحركات المعقدة (التبييت، الترقية، الفحص، الكش ميت)
- ✅ كشف الفحص والكش ميت والتعادل
- ✅ تاريخ الحركات الكامل

### الذكاء الاصطناعي
- 🤖 ذكاء اصطناعي قوي جداً باستخدام Minimax + Alpha-Beta Pruning
- 🎯 5 مستويات صعوبة (سهل، متوسط، صعب، خبير، إلهي)
- ⚡ تقييم موضعي متقدم مع جداول الموضع
- 🧠 كاش ذكي لتحسين الأداء

### الميزات المتقدمة
- 🎨 واجهة كلاسيكية فخمة بألوان ذهبية وسوداء وبيضاء
- ⏱️ ساعة شطرنج مع عداد وقت ديناميكي
- 🔊 تأثيرات صوتية متقدمة (حركة، التقاط، فحص، كش ميت)
- 📊 إحصائيات اللعبة والنتائج
- 👥 نظام غرف متعددة للعب مع الأصدقاء
- 🎛️ إعدادات اللعبة والصوت

## 🎮 كيفية اللعب

### ضد الذكاء الاصطناعي
1. اختر مستوى الصعوبة (سهل، متوسط، صعب، خبير)
2. اختر لونك (أبيض أو أسود)
3. ابدأ اللعبة وقم بتحريك القطع

### مع الأصدقاء
1. أنشئ غرفة جديدة
2. شارك كود الغرفة مع صديقك
3. دع صديقك ينضم باستخدام الكود
4. ابدأ اللعبة

## 🛠️ التكنولوجيات المستخدمة

- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4 + shadcn/ui
- **State Management**: React Hooks
- **Routing**: Wouter
- **Build Tool**: Vite
- **Chess Engine**: محرك شطرنج مخصص
- **AI**: Minimax Algorithm + Alpha-Beta Pruning

## 📊 خوارزميات الذكاء الاصطناعي

### Minimax Algorithm
- تقييم جميع الحركات الممكنة
- اختيار أفضل حركة بناءً على التقييم
- عمق البحث: حتى 5 مستويات

### Alpha-Beta Pruning
- تحسين كفاءة Minimax
- قطع الفروع غير الضرورية
- تقليل الحسابات بنسبة كبيرة

### Position Evaluation
- تقييم المادة (قيمة القطع)
- تقييم الموضع (جداول الموضع المتقدمة)
- تقييم الأمان (حماية الملك)
